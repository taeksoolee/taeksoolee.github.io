## SNMP (Simple Network Message Protocol) * application layer * 관리자 : manager, 관리대상 : agent * TCP/IP 기반 프로토콜 * MIB에 정보를 수집, 관리, 수정(동작을 변경) * 중요 요소 : 자원, 성능, 장애 # Message 분류 (PDU) >> getRequest : 변수값을 리딩 >> getNextRequest : 변수의 다음 변수값을 리딩 >> setRequest : 변수값을 변경 >> getResponse : 값 전단달 (Agent > Master) >> trap : 특정상황 발생을 전달 (Agent > Master) # 라이브러리 # java: snmp4j
