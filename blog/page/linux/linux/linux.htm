## 서버 시간(timezone) 변경 * 참고 : https://twpower.github.io/95-set-ubuntu-timezone # ubuntu // 국가 확인 (/usr/share/zoneinfo/Asia/Seoul) $ cd /usr/share/zoneinfo/ // 1번 방법 $ sudo ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime ## systemctl : 시스템 환경에 대한 기능을 제공 # 실행 확인 $ sudo systemctl status [프로세스명] ## 변수 설정( export ) $ export [변수명]=[값] $ export VARIABLE="value" $ echo $VARIABLE => vlaue ## network (네트워크) 명령어 # net-tools (네트워크 명령어 모음) 설치 # ping : 연결상태 확인 // 구글 DNS 서버로 ping 요청 = 서버 작동하는지 확인 $ ping 8.8.8.8 # route -n : 라우트 테이블 출력 (라우트) * Destination : IP * Gateway : gateway 주소 * Genmask : subnetmask * Flugs * Metric : 우선숭뉘 * Ref * user * Iface : 인터페이스 $ route -n // default(0.0.0.0) ip에 192.168.10.1 게이트웨이, enp0s3 인터페이스 추가 $ route add -net default gw 192.169.10.1 dev enp0s3 // default(0.0.0.0) ip에 192.168.10.1 게이트웨이, enp0s3 인터페이스 삭제 $ route del -net default gw 192.169.10.1 dev enp0s3 # ifconfig : ip 주소 정보를 출력 # nmap // 설치 $ sudo apt install nmap # netstat ## 용량확인 $ du -sh [파일이름]
