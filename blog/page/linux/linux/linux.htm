## 리눅스 # 경로 /       : root ./      : 현재 ../     : 부모 ~/     : home $     : 마지막을 의미하는 경우가 많음 # 파일형태 파일형태    user    group            other -                      wrx             wrx             wrx -  : file d : directory l : link w : write ( 쓰기권한) r: read (읽기 권한) x : execute (실행 권한) ## 리스트 명령어 * 참고 : https://itholic.github.io/linux-basic-command/ ls -l(리스트 형태로 보여줌) -a(모두 보여줌) -i(인덱스를 보여줌) ## 디렉토리 / 파일 명령어 # 디렉토리 생성 $ mkdir [디렉토리 명] -p (--parents) 상위 경로 생성 -m (--mode=MODE) 권한 설정 (기본 755) -v (--verbose) 디렉토리 생성후 메시지 출력 # 파일 생성 $ touch [파일명] # 파일 / 디렉토리 삭제 $ rm -f [파일명] // * 완전 삭제(주의 할것) $ rm -rf [디렉토리 명] # 복사 $ cp -rp [복사할파일] [복사될 파일] -r 디렉토리 이하파일 -p 메타 데이터 함께 복사 // 대용량 파일복사 $ cp -rf [source] [target] $ rsync -anv --progress [sorce] [target] # 이동 $ mv [이동할파일] [이동될 파일] ## 권한 관련 명령어 # 폴더 또는 파일 주인변경 $ chown # 폴더 또는 파일 권한변경 $ chmod ## tail 명령어 $ tail -F [파일명] -F : 실시간 반영 ## link 파일 만들기 ## 서버 시간(timezone) 변경 * 참고 : https://twpower.github.io/95-set-ubuntu-timezone # ubuntu // 국가 확인 (/usr/share/zoneinfo/Asia/Seoul) $ cd /usr/share/zoneinfo/ // 1번 방법 $ sudo ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime
