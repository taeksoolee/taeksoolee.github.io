## 리스트 명령어 * 참고 : https://itholic.github.io/linux-basic-command/ ls -l(리스트 형태로 보여줌) -a(모두 보여줌) -i(인덱스를 보여줌) ## 디렉토리 / 파일 명령어 rm -rf 디렉토리 완전 삭제(주의 할것) cp -rp "복사할파일" "복사될 파일" -r 디렉토리 이하파일 -p 메타 데이터 함께 복사 mv "이동할파일" "이동될 파일" chown : 폴더 또는 파일 주인변경 chmod : 폴더 또는 파일 권한변경 ## 리눅스 # 경로 /       : root ./      : 현재 ../     : 부모 ~/     : home $     : 마지막을 의미하는 경우가 많음 # 파일형태 파일형태    user    group            other -                      wrx             wrx             wrx -  : file d : directory l : link w : write ( 쓰기권한) r: read (읽기 권한) x : execute (실행 권한) ## 리눅스 대용량 파일 복사 cp -rf [source] [target] rsync -anv --progress [sorce] [target]
