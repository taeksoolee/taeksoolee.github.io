## 계정 목록 # 전체목록 $ cat /etc/passwd # bash 사용자목록 $ grep /bin/bash /etc/passwd ## 명령어 # 계정 생성 * 레드햇 계열에서는, useradd 명령어로 생성되는 일반 사용자 계정의 UID가 500 이상 // home 디렉토리 생성 등 기본 설정 포함 명령 $ sudo adduser [username] // 계정만 생성 명령 $ sudo useradd [username] # 계정 관리자 권한 부여 $ sudo adduser [username] sudo # 계정 삭제 // home 디렉토리 생성 등 기본 설정 포함 명령 $ sudo deluser [username] // 계정만 삭제 명령 $ sudo userdel [username] ## 권한 관련 명령어 # 폴더 또는 파일 주인변경 $ chown # 폴더 또는 파일 권한변경 $ chmod
