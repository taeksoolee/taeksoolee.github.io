## 기본 명령어 # git repository 초기화 (현재 디렉토리를 repository로) $ git init // repository 선택 $ git remote [리모트 주소] # git repository 복사 (디렉토리 생성) $ git clone [리모트 주소] ## 저장소 저장 // stage 저장 $ git add . // HEAD 추가 - local repository(.git) 저장 $ git commit -m "메시지 입력" // git 저장소에 저장 $ git push [remote] [branch] ## 저장내역 확인 // add file 확인 $ git status // commit 내역 - local repository(.git) 확인 $ git log ## branch - checkout # 브랜치 추가 (가져오기) $ git branch [브랜치명] # 브랜치 변경 $ git checkout [브랜치명] ## branch ## merge ## config 설정 명령어 # 성정 목록 확인 $ git config --global --list # 설정 수정 $ git config --global user.name [사용자명] $ git config --global user.email [이메일] ## windows 자격 증명 수정 * 제어판 > 사용자계정 > 자격증명관리 > windows 자격증명 > github 항목 삭제 후 수정
