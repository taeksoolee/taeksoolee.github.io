## C's history # K&R C(Classic C) : 유닉스와 함께 제공된 라이브러리 # ANCI(American National Standard Institude) C89 / ISO(International Organization of Standarddization) C90 The Spirit of C 프로그래머를 믿어라 프로그래머가 필요한 일을 하는것을 방해하지마라(책임도 함께 지게하라) 언어를 작고 단순하게유지해라 한 연산을 할때 한가지 방법만 제공해라 호환을 보장할 수 없더라도 속도를 빠르게해라 # ANCI / ISO C9X 국제적인 문자집합추가 64비트 프로세서 대응 과학 공학수치 계산 적합도 개선(FORTRAN 대체) 언어를 단순하게 유지 커미티는 (c가 아니라) C++가 크고 야심찬 언어가 되는것을 허용하는 것을 동의 # The C11 standard C99의 일부를 선택적인 기능으로 변경 멀티 프로세서 대응 concurrent programming # The C18 Standard 새로운 기능 추가 없이 C11의 결함 수정
